# Spotify Top 50 World Analytics

## Business Problem

The global music industry is highly competitive, with thousands of songs released every day. Understanding what drives song popularity, artist performance, and chart rankings is crucial for music platforms, artists, and record labels.

This project aims to analyze Spotify's Top 50 World dataset to uncover trends in song popularity, chart performance, explicit content, album characteristics, and artist dominance.

---

## Project Objectives

* Analyze global music trends using Spotify Top 50 data.
* Identify top-performing songs and artists.
* Explore factors influencing song popularity.
* Examine chart rankings and artist dominance.
* Build an interactive dashboard for business insights.

---

## Dataset

The dataset contains information about Spotify Top 50 songs worldwide, including:

* Date
* Chart Position
* Song Name
* Artist
* Popularity Score
* Duration
* Album Type
* Total Tracks
* Release Date
* Explicit Content Flag

## Tools & Technologies Used

* **PostgreSQL** – Data storage and analytical SQL queries
* **Power BI** – Interactive dashboard creation and visualization
* **Generative AI (ChatGPT)** – Assisted in SQL query generation, DAX measure creation, and insight storytelling
* **GitHub** – Project documentation and version control

---

## Project Workflow

1. Imported and explored Spotify Top 50 dataset.
2. Performed SQL-based analysis using PostgreSQL.
3. Wrote analytical queries using:

   * CTEs
   * Window Functions
   * Subqueries
   * Aggregations
4. Created DAX measures for KPI analysis in Power BI.
5. Designed an interactive dashboard to visualize trends and insights.
6. Generated business insights from the analysis.

---

## Key Insights

* Identified the most popular songs and artists globally.
* Analyzed explicit vs non-explicit song distribution.
* Examined artist dominance across charts.
* Explored relationships between popularity, duration, and album characteristics.
* Tracked chart rankings and music trends over time.
